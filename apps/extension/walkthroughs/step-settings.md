# Configure Your Preferences

In the **Settings** tab, customize how Claude Code will work with your project.

| Setting | Options |
|---------|---------|
| **Type Strictness** | Strict / Moderate / Loose |
| **Paradigm** | Functional / OOP / Mixed |
| **Comment Style** | Minimal / JSDoc / None |
| **Autonomy Level** | Ask-first / Proceed / Autonomous |
| **Generated language** | English / Korean |

Settings are saved to `.vscode/` and persist across sessions.
