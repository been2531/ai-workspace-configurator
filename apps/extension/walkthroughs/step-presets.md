# Apply a Community Preset

In the **Presets** tab, browse and apply presets before generating.

**Built-in presets**

| Preset | What it changes |
|--------|----------------|
| Karpathy Coding Principles | AGENTS.md — think, simplicity, surgical changes, goal-driven |
| Minimal | AGENTS.md — three-rule autonomous mode |
| Strict TypeScript | CLAUDE.md — no `any`, Result pattern, pure functions |

**GitHub presets**

Any public GitHub repo tagged `claude-code` appears here. Search by name or browse by stars.

Select a preset → click **⚡ Generate** to apply it on top of your detected stack.
