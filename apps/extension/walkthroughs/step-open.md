# Open Your Project

Open the folder you want to configure with AI coding assistants.

AI Workspace Configurator detects your stack automatically:

| Detected | Examples |
|----------|---------|
| Language | TypeScript, Python, Go, Rust, Java, … |
| Framework | Next.js, Django, FastAPI, NestJS, … |
| Package manager | npm, pnpm, yarn |

**The extension activates on startup.** On first open, it will offer to generate files immediately.
