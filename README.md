## What it does

**AI Workspace Configurator** detects your project stack (language, frameworks, package manager) and instantly writes a set of AI instruction files tailored to it — so Claude Code, Cursor, and Codex understand your codebase from the very first prompt.

| Generated file | Purpose |
|----------------|---------|
| `CLAUDE.md` | Stack-specific agent rules for Claude Code |
| `AGENTS.md` | Multi-agent handoff protocol + reviewer checklist |
| `.cursorrules` | Cursor AI rules (legacy format) |
| `.cursor/rules/project.mdc` | Cursor AI rules (new MDC format) |
| `.mcp.json` | MCP server auto-configuration |
| `.claude/skills/<name>/SKILL.md` | Slash commands (Agent Skills format): `/run`, `/test`, `/review`, `/commit`, `/fix`, `/pr`, `/security`, `/doctor`, plus `/type-check`, `/db-migrate`, `/deploy` per stack |

Everything runs **100% locally** — no API calls, no data sent anywhere.

Meta-framework priority: Next.js > Nuxt > SvelteKit > Remix (prevents duplicate entries like React + Next.js).

---

### Rich generated content

`CLAUDE.md` includes:
- **Project structure guide** — stack-aware directory layout (Next.js App Router, Django apps, Go packages, Rust modules, …)
- **Testing conventions** — per-stack patterns (pytest fixtures, Vitest, Go table tests, Rust `#[cfg(test)]`)
- **AI workflow hints** — actionable steps for TypeScript type-checking, Prisma migrations, linter runs
- **Framework rules** — 5–8 specific rules per detected framework

`AGENTS.md` includes a **stack-specific reviewer checklist** covering TypeScript, NestJS, Django, FastAPI, Prisma, Drizzle, Firebase, Go, and Rust.

---

### Stale config detection

On every workspace open, the extension checks whether `CLAUDE.md` was generated by this tool and compares the current stack against the saved metadata. If new frameworks were added since the last generation, a warning appears:

> _"AI Workspace files may be outdated (new: Prisma, Vitest). Regenerate?"_

Three distinct startup states:
1. **No config files** → offer to generate
2. **Outdated config** → yellow warning with a description of what changed
3. **Up-to-date** → gentle info notification with regenerate / open dashboard options

## Quick start

1. Install **AI Workspace Configurator** from the [VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=ai-workspace-configurator.ai-workspace-configurator)
2. Open any project folder in VS Code
3. Click **"⚡ Generate Now"** in the startup notification
   — or run **"AI Workspace: Generate Config Files"** from the Command Palette (`Ctrl+Shift+P`)
4. Files appear in your workspace root — start Claude Code immediately

## CLI

```bash
npm install -g @ai-workspace-configurator/cli

cd my-project
ai-workspace init          # detect stack and generate files
ai-workspace profile       # set your coding style preferences
ai-workspace search react  # browse community presets
ai-workspace apply <id>    # apply a preset
ai-workspace publish       # share your config as a preset
```

## MCP server (Claude Code native)

Add to `.mcp.json`:

```json
{
  "mcpServers": {
    "ai-workspace": {
      "command": "npx",
      "args": ["@ai-workspace-configurator/mcp"]
    }
  }
}
```

Then use directly in Claude Code chat:

```
Generate AI workspace config for this project
Apply the karpathy preset
Search community presets for TypeScript
```

